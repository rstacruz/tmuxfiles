# Key reference

## Tabs

| Key               | Description   |
| ----------------- | ------------- |
| `^a ^c`           | New tab       |
| `^a ^x`           | Close tab     |
| `^a ^n` / `^a `p` | Next/prev tab |
| `^a ,`            | Rename tab    |
| `^a {0..9}`       | Switch to tab |
| `^a ^w`           | Switch to...  |

## Panes

| Key                       | Description          |
| ------------------------- | -------------------- |
| `^a ^s` (default: `^a "`) | Split horiz          |
| `^a ^v` (def: `^a %`)     | Split vert           |
| `^a o`                    | Focus next pane      |
| `^a q`                    | Focus pane...        |
| `^a {`                    | Move pane            |
| `^a !`                    | Move pane to new tab |

## Sessions

| Key    | Description    |
| ------ | -------------- |
| `^a #` | New session    |
| `^a $` | Rename session |
| `^a *` | Close session  |

## Other shortcuts

| Key    | Description       |
| ------ | ----------------- |
| `^a ?` | Help              |
| `^a b` | Toggle status bar |
| `^a I` | Install plugins   |
| `^a R` | Reload config     |

## Copying

| Key    | Description                 |
| ------ | --------------------------- |
| `^a [` | Scroll mode (exit with `q`) |

`space` to select in scroll mode, then `Enter` to copy to clipboard

## Fingers

| Key    | Description      |
| ------ | ---------------- |
| `^a F` | Copy url/path... |

Shows hints for things to copy
